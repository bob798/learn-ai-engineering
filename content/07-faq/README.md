# 07 · FAQ · 常见 QA

> 不是「AI 是什么」的科普 FAQ，而是**一个工程师天天用 AI 干活时，反复撞上的那些「为什么会这样、后续怎么办」**。
>
> 每一条都来自真实场景：先讲现象，再用一个脱敏的真实例子坐实，然后抬到「这在 AI 领域叫什么问题、通用解法是什么、有没有更省力的做法」。目标是把零散的踩坑，变成下次能直接套用的判断。

## 这个栏目怎么用

- **遇到类似困惑时**：当成「别人也踩过、已经想清楚」的速查表，直接看结论和「怎么做」。
- **想搞透机制时**：每篇都标了它在 AI 领域的学名（满足即止 / Self-Refine / Reflexion …），顺着关键词能往学术原文挖。
- **想固化成习惯时**：每篇末尾的「更简便的方式」给的是可以写进 `CLAUDE.md` / 记忆 / 团队规约的常驻规则，而不是一次性技巧。
